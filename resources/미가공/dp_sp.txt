# PX - pixel

화면을 구성하는 최소 단위로 전체 화면에서의 절대적 크기를 나타내는 단위이다

## 상대적 단위

안드로이드의 디바이스는 해상도, 화면 크기, 비율 등에서 매우 다양하다.
따라서 화면 크기가 변해도 절대적 크기는 같은 DP,SP등의 단위가 필요하다

## DP - Density independent Pixels

밀도가 다른 화면에서 같은 크기로 나타나는 **크기의 측정단위**이다.
`160dpi`(dots-per-inch,화면 밀도 단위) 기준 1픽셀과 거의 동일한 크기를 가지고 있는 가상의 단위이다.
px = dp * (dpi / 160) 의 식으로 실제 픽셀 크기를 가늠해 볼 수 있다.
코드에서는 

```Kotlin
val scale: Float = resources.displayMetrics.density
```
의 꼴로 dp에 곱할 (dpi/160) 값을 얻을 수 있다.

## SP - Scalable Pixels

**텍스트 크기**를 나타낼 수 있으면서 dp처럼 화면 밀도에 관계없이 일정한 크기를 유지하는 단위이다.
사용자별로 글씨 크기 설정이 다르다. 따라서 sp를 통해서 dp를 글씨 크기에 대해 **확장 대응**하게 해야 한다.
